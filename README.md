# Teamarr v2

Provider-agnostic sports data layer for EPG generation.

## Quick Start

```bash
# Create virtual environment
python3 -m venv .venv
source .venv/bin/activate

# Install with dev dependencies
pip install -e ".[dev]"

# Run tests
pytest
```

See [DEVELOPMENT.md](DEVELOPMENT.md) for detailed setup instructions.
